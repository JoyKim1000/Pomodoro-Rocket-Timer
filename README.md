# 🚀 뽀모도로 목표 성공 로켓 (Success Rocket)

> **"대표님의 집중력이 성공의 연료가 되어 로켓을 쏘아 올립니다!"**  
> AI 1인 기업 대표님들을 위한 감성 충만, 생산성 폭발 뽀모도로 타이머입니다.

![Mascot](success_rocket.png)

## ✨ 주요 특징 (Key Features)

*   **🚀 성공 로켓 발사 시스템:** 25분 집중 미션 완료 시 로켓이 실제로 발사되는 역동적인 애니메이션!
*   **📊 주간 성공 궤도 리포트:** 월요일부터 일요일까지의 집중 시간을 시각화하여 성취감 극대화.
*   **🍵 감성적인 티타임 일시정지:** '멈춤' 대신 '차 한 잔의 여유'를 제안하는 따뜻한 UI.
*   **🔔 스마트 알림 관제 센터:** 브라우저 탭 타이틀 업데이트, 푸시 알림, 기분 좋은 오디오 알람 지원.
*   **📱 PWA 모바일 최적화:** 스마트폰 홈 화면에 설치하여 언제 어디서든 앱처럼 사용 가능.

## 🛠 기술 스택 (Tech Stack)

*   **Frontend:** Vanilla JavaScript, HTML5, CSS3 (Custom Neon Design)
*   **Aesthetics:** Futuristic Glassmorphism & Space Theme
*   **PWA:** Service Worker & Web App Manifest integration

## 🚀 시작하기 (How to Use)

1.  본 저장소를 클론하거나 파일을 다운로드합니다.
2.  `index.html` 파일을 브라우저로 엽니다.
3.  **[엔진 점화]** 버튼을 눌러 미션을 시작하세요!
4.  시간이 종료되면 로켓 발사와 함께 스마트 알람이 울립니다.

## 📱 모바일 설치 (Mobile Installation)

1.  배포된 주소에 접속합니다.
2.  **iOS:** Safari 공유 버튼 -> '홈 화면에 추가'
3.  **Android:** Chrome 메뉴 -> '홈 화면에 추가' 혹은 '앱 설치'
